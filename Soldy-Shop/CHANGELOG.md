# Changelog

All notable changes to this project are documented in this file.

## [1.0.1] - 2026-06-15
### Added
- Contact endpoint support with new backend contact route and controller.
- Legal pages: Privacy Policy and Terms & Conditions.
- Frontend environment example file (`.env.example`).
- Explore navbar account dropdown actions: Profile, My Orders, Settings, Admin Panel (admin only), Logout.

### Changed
- Improved frontend UX across Home, Explore, Product, Cart, Checkout, Profile, Orders, Settings, and Wishlist flows.
- Improved responsive behavior for admin dashboard/layout pages across mobile, tablet, and desktop.
- Updated shared components including navbar, footer, branding, product cards, and ratings visuals.
- Updated auth/cart slice integration details and image URL handling.
- Updated backend auth, upload, email utility, and server wiring.
- Updated frontend/backend lockfiles and generated frontend production assets.

### Notes
- This release is a broad UI/UX and platform polish update.
- No database migration was introduced in this release.
